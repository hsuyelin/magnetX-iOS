## Installation

RxSwiftX is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'RxSwiftX'
```

### Optional

```ruby
pod 'RxSwiftX/DataSources'
pod 'RxSwiftX/Network'
pod 'RxSwiftX/MJRefresh'
```
